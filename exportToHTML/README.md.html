1    # GRU4Rec_TensorFlow
2    TensorFlow implementation of *GRu4Rec*, which was descibed in "Session-based Recommendations With Recurrent Neural Networks". See paper: http://arxiv.org/abs/1511.06939. 
3    
4    # Requirements
5    Python: 2.7
6    
7    Pandas < 0.17 
8    
9    Numpy 1.12.1 or later
10   
11   TensorFlow: 0.12.1
12   
13   # Usage
14   Train/Test file should consists of three columns:   
15   
16        First column: SessionId  
17        Second column: ItemId  
18        Third column: Timestamps
19   
20   To train a model with default parameter settings:
21   
22       $ python main.py    
23   
24       Other optional parameters include:   
25        --layer: Number of GRU layers. Default is 1.  
26        --size: Number of hidden units in GRU model. Default is 100.   
27        --epoch: Runing epochs. Default is 3.   
28        --lr : Initial learning rate. Default is 0.001.   
29        --train: Specify whether training(1) or evaluating(0). Default is 1.   
30        --hidden_act: Activation function used in GRU units. Default is tanh.   
31        --final_act: Final activation function. Default is softmax.    
32        --loss: Loss functions, cross-entropy, bpr or top1 loss. Default is cross-entropy.      
33        --dropout: Dropout rate. Default is 0.5.
34   
35   To evaluate a trained model:
36   
37       $ python main.py --train 0
38       
39       One optional parameter is:    
40        --test: Specify which saved model to evaluate(only used when --train is 0). Default is 2.
41
